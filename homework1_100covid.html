Downloading the FASTA file¶

Comecei por ir ao NCBI e procurar por "Severe acute respiratory syndrome coronavirus" e dei 'sort' pela sua data de submisso. De seguida selecionei os 100 FASTAS mais recentes e fiz download do FASTA inteiro com todas as suas sequncias.

O ficheiro FASTA que continha as 100 sequncias foi dividido posteriormente utilizando a ferramenta split-fasta.

Para instalar a ferramenta no terminal:

pip install split-fasta

Este pacote permite dividir o nosso FASTA integro em pequenos ficheiros FASTAS cada um contendo a sua sequncia.

import fileinput #mdulo para lidar com os ficheiros
import pandas as pd #mdulo do pandas para lidar com os dados (foi mais fcil para mim)
import os #para lidar com os ficheiros no sistema operativo

#criar dataframe para guardar os dados
dados_covid = pd.DataFrame() 

#criar uma lista automtica com todos os ficheiros presentes no folder
lista_ficheiros = os.listdir('/Users/filipecaramelo/Documents/Msc Bioinformatics/Data Structure/Homework/1st Homework/100covid_split_files')

#criar automticamente o nmero do ficheiro
nr_ficheiro=0

#automatizao do processo para todos os ficheiros
for i in lista_ficheiros:
    contadores={'A':0,'C':0,'G':0,'T':0,'\n':0,'N':0} #criao do contador
    caminho='/Users/filipecaramelo/Documents/Msc Bioinformatics/Data Structure/Homework/1st Homework/100covid_split_files/%s' % i #automatizar o caminho para cada ficheiro individual
    #automatizao para a leitura de cada linha e contagem nucleotidica
    with fileinput.input(files=(caminho)) as f:
        f.readline()
        for line in f:
            for c in line:
                contadores[c] += 1
    nr_ficheiro += 1 #automatizao do nr de ficheiro
    #introduo desta nova contagem no dataframe
    dados_covid = dados_covid.append({'Ficheiro': nr_ficheiro,'A':contadores['A'], 'T':contadores['T'], 'G':contadores['G'], 'C':contadores['C'], 'N':contadores['N']}, ignore_index = True)
