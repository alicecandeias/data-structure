Downloading the FASTA file¶

Comecei por ir ao NCBI e procurar por "Severe acute respiratory syndrome coronavirus" e dei 'sort' pela sua data de submissão. De seguida selecionei os 100 FASTAS mais recentes e fiz download do FASTA inteiro com todas as suas sequências.

O ficheiro FASTA que continha as 100 sequências foi dividido posteriormente utilizando a ferramenta split-fasta.

Para instalar a ferramenta no terminal:

pip install split-fasta

Este pacote permite dividir o nosso FASTA integro em pequenos ficheiros FASTAS cada um contendo a sua sequência.

(Provavelmente haverá algo mais fácil para resolver este problema, já que não consegui ler o ficheiro inteiro utilizando o código do professor.)

import fileinput #módulo para lidar com os ficheiros
import pandas as pd #módulo do pandas para lidar com os dados (foi mais fácil para mim)
import os #para lidar com os ficheiros no sistema operativo

#criar dataframe para guardar os dados
dados_covid = pd.DataFrame() 

#criar uma lista automática com todos os ficheiros presentes no folder
lista_ficheiros = os.listdir('/Users/filipecaramelo/Documents/Msc Bioinformatics/Data Structure/Homework/1st Homework/100covid_split_files')

#criar automáticamente o número do ficheiro
nr_ficheiro=0

#automatização do processo para todos os ficheiros
for i in lista_ficheiros:
    contadores={'A':0,'C':0,'G':0,'T':0,'\n':0,'N':0} #criação do contador
    caminho='/Users/filipecaramelo/Documents/Msc Bioinformatics/Data Structure/Homework/1st Homework/100covid_split_files/%s' % i #automatizar o caminho para cada ficheiro individual
    #automatização para a leitura de cada linha e contagem nucleotidica
    with fileinput.input(files=(caminho)) as f:
        f.readline()
        for line in f:
            for c in line:
                contadores[c] += 1
    nr_ficheiro += 1 #automatização do nr de ficheiro
    #introdução desta nova contagem no dataframe
    dados_covid = dados_covid.append({'Ficheiro': nr_ficheiro,'A':contadores['A'], 'T':contadores['T'], 'G':contadores['G'], 'C':contadores['C'], 'N':contadores['N']}, ignore_index = True)
